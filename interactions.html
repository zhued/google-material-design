Interaction is one of the most important yet simple concepts within Material Design. It's essentially how the user will see their input come to life, a crucial moment for any designed object with a user base. The connection between the user and the screen must become as realistic and intuitive as possible for the best results.

Input Reactions

We touch our screens constantly throughout the day, giving constant input to our small machines. With every interaction, there is a small detail letting the user know that their touch as been received.

It may be a slight detail, but it makes all the difference to feel that reaction coming back to the user. This feedback makes the user more involved with their applications, as if they're communicating with each other.

Material Response

For cases where the reponse if more intricate than a simple ripple, there is always a point of origin correlating with every touch. Objects cannot be flying in randomly, that would be confusing and unnatural for the user's experience. Where ever the user touches, that is the point of origin from where that object will pop out and come alive, making the interaction seem real.
